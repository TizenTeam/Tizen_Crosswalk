Milestone Santa
Release Date: 27 Dec 2013.
========================================================================

Based on Chromium-
Version: 32.0.1700.69.
Channel: Beta.
Revision: 6f5eb8456ae3021c7396be3a67c454d82b2cfb30.

---------------------------------------------------------------------

New Features enabled in this release:
1) WebGL.
2) MultiWindow support.
3) FullScreen support.

---------------------------------------------------------------------

Fixed Issues:
1) [#129] Position the Child Window/Popups correctly.
2) [#145] content_shell SIGSEGV, while trying to open a context menu.
3) [#146] Fix IPCHandShake.
4) [#151] Crash when resizing browser window.
5) [#154] Fix FullScreen bounds.
6) [#169] Fix heap and stack corruption in DisplayChannelHost.
7) Support AddressSanitizer tool.
   http://www.chromium.org/developers/testing/addresssanitizer
8) Support Component=shared_library  build.

Lot more enhancements.

---------------------------------------------------------------------

Known Issues:
1) Dialogs are not supported.(i.e. File picker)
2) Keyboard: CapsLock and special characters are not recognized correctly.
3) Missing Drag and Drop support.
4) Browser window cannot be resized(Only FullScreen mode is supported).
